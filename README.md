# IssueTracker 


## how to run
1. install dependencies:

    `npm i -g pnpm`

    `pnpm i`

2. run:

    * In memory:

      run: `pnpm dev:all-hook` and open browser on : `localhost:8082`


    * In file system: 

      1. build: `pnpm build`
    
      2. run: `pnpm start`
